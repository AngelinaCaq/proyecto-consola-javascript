# Mini juegos en JavaScript

Este proyecto consiste en una pequeña colección de juegos desarrollados con JavaScript que se ejecutan en el navegador utilizando `prompt` y `alert` para la interacción con el usuario.

El objetivo del proyecto es practicar conceptos fundamentales de JavaScript como funciones, condicionales, bucles, arrays y objetos.

## Juegos incluidos

### 1. Cachipún (Piedra, Papel o Tijera)

El jugador compite contra la consola durante cinco rondas.
En cada ronda se comparan las elecciones y se actualiza el puntaje.
Al final del juego se muestra el resultado total.

### 2. Adivina el número

El jugador tiene cinco intentos para adivinar un número.
Después de cada intento el programa indica si el número es mayor o menor.

### 3. Aventura

Pequeño juego de decisiones donde el jugador elige un camino.
Dependiendo de la elección se obtiene uno de los posibles finales.

## Conceptos de JavaScript utilizados

* prompt() y alert()
* Condicionales (if / else)
* Bucles (for)
* Funciones
* Arrays
* Objetos
* Math.random()
* Validación de datos con isNaN()

## Cómo ejecutar el proyecto

1. Clonar o descargar este repositorio.
2. Abrir el archivo `index.html` en un navegador.
3. Elegir uno de los juegos desde el menú inicial.
